PROJECT: CAMPUS ISSUE DESK
HOW TO RUN:
1. Ensure Node.js is installed.
2. Open the "student-portal-backend" folder in a terminal.
3. Run command: npm install
4. Run command: node server.js

5. Open "index.html" in your browser.
